# R-Project

This repository contains a collection of R-based data analysis projects completed as part of the Undergraduate R Programming Course in the Department of Statistics, Noakhali Science and Technology University (NSTU).

## Overview
These projects were developed to strengthen practical understanding of statistical computing, data visualization, and applied data analysis using the R programming language. Each folder represents an individual project focused on different real-world datasets and analytical methods.

## Projects Included
- Auto Data Project – Analyzing automobile dataset and building regression models.  
- Bank Data Project – Statistical insights from bank marketing campaign data.  
- COVID-19 Data Project – Time series and trend analysis of COVID-19 data.  
- Econometrics in R – Implementation of econometric models in R.  
- Google Trend Data – Exploring search pattern data from Google Trends.  
- RailTrail Project – Exploratory analysis and regression modeling.  
- Regression Project – Demonstration of linear and multiple regression techniques.  
- Spydata Project – Data manipulation and summary statistics practice.  
- Yahoo Finance Data Time Series – Time series forecasting using financial data.  
- mtcars Project – Classic dataset analysis for regression and correlation studies.  

## Tools and Libraries
- R Base  
- tidyverse  
- ggplot2  
- dplyr  
- forecast  
- readr  
- lubridate  

## Purpose
The main objectives of this project series were to:
- Apply theoretical statistical knowledge to real datasets.  
- Learn efficient data handling and visualization in R.  
- Build a foundation for advanced data analysis and research.
